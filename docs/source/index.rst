Документация CPU Version
========================

Добро пожаловать в документацию проекта CPU Version!

Этот проект предоставляет инструменты для запуска инференса обученной модели
Detectron2 на CPU без необходимости использования GPU. Проект включает:

* Модуль для запуска инференса на изображениях
* Обнаружение объектов и сегментация
* Анализ перекрывающихся детекций
* Конвертация координат
